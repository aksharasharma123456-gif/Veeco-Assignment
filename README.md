# Sigma Live — Supply Chain & Demand Planning
**Veeco New Graduate Engineering Assignment | Track 1**

A dual-application React ecosystem simulating a live Snowflake + Sigma Computing environment for supply chain operations.

## Architecture
- **Component A — Sigma Live Insight**: Analytics dashboard with real-time KPIs, inventory vs. demand charts, and supplier bottleneck visibility
- **Component B — Sigma Live Action**: Operational input app for forecast overrides, supply update logging, and transit delay tracking
- **Shared State Layer**: React Context API simulating Snowflake's single source of truth — changes in Action instantly reflect in Insight

## Tech Stack
React · Vite · Recharts · React Context API · CSS3
